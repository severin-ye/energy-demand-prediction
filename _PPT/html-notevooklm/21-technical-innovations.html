🎯 Contextual Slate Bandit 기반 Top-K 추천

주어진 컨텍스트 c 하에서, 후보 집합으로부터 Top-K 방안 목록(Slate) 선택: S = [π₁, π₂, ..., πₖ]

다목적 보상 함수:
r(π) = α·AcceptIntent + β·Like - γ·Dislike + λ·ΔPeak + μ·ΔEnergy - ρ·Cost - η·RejectionPenalty
학습 알고리즘

Thompson Sampling / LinUCB 채택, 다양한 컨텍스트에서 고수익 방안 구조 학습

탐색-활용 균형

최소 탐색 비율(20%) 유지, Action Masking을 통해 안전성 보장

실행 및 피드백 파이프라인

신호 융합 컨텍스트, 예측 신뢰도 및 비즈니스 제약을 bandit 상태로 결합하여 Slate 구성 주도.
행동 필터링 실행 불가/고위험 행동 Mask; 주요 지표 임계값 초과 시 안전 등급 하향 전략 트리거.
피드백 귀인 실행 후 수락, 거부 및 피크 변화 지표 수집, 보상 가중치 업데이트 및 설명 가능한 특징 기록.
