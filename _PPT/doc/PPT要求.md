角色设定（Role）
你是一名计算机科学/人工智能方向的研究生，正在为导师准备一份**论文方法复现（Reproduction Study）**的学术汇报 PPT。

任务目标（Goal）
请基于指定论文，生成一份论文级别（paper-level）的复现 PPT，要求：

方法（Method）与论文保持高度一致

实验设置（Experimental Setup）与论文严格对齐

结果（Results）在合理误差范围内与论文结果一致

PPT 不是“最低限度复现”，而是最大程度还原论文方法与结论

PPT 结构要求（Structure）

1. Introduction

研究背景与问题定义

论文研究动机（Why this work matters）

论文的核心贡献（Contributions）

2. Related Work（如论文中有）

简要说明论文在该领域中的位置

与相近方法的关键差异（只保留与本文强相关内容）

3. Method (严格对齐论文)

使用与论文完全一致的结构顺序

明确说明：

模型结构 / 算法流程

关键模块设计

重要公式（如有）

所有方法描述需以“论文原方法”为基准，而非简化版本

4. Experimental Setup

数据集（Dataset）

数据来源

数据规模

划分方式

实验设置

训练 / 推理参数

硬件与软件环境（如论文中有）

评价指标（Metrics）

与论文保持一致

明确指标含义

5. Results (论文对齐复现结果)

使用与论文相同或高度相似的表格/图表结构

对比展示：

论文原始结果

本次复现结果

强调：

一致的趋势

可解释的偏差（如存在）

6. Analysis / Discussion

对结果进行分析：

哪些部分复现得较好

哪些部分存在偏差

简要讨论可能原因（数据、参数、实现细节等）

7. Issues & Insights（非常重要）

Issues（复现中发现的问题）

难以完全复现的环节

方法在现实或不同设置下的局限

Insights / Ideas（个人研究思考）

对论文方法的理解深化

潜在改进方向

后续研究可能性

⚠️ 该部分用于展示研究思维，而非批评论文

8. Conclusion

总结复现工作成果

复现是否支持论文结论

本次复现对未来研究或应用的启示

风格与形式要求（Style）

学术风格、正式表达

面向导师汇报

逻辑清晰、层次分明

避免“作业感”，强调“研究复现”与“方法理解”

输出形式（Output）

以「PPT 每一页 = 一个 Slide」的形式输出

每一页包含：

Slide 标题

关键要点（bullet points）
